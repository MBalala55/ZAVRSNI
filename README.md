# ZAVRŠNI RAD

## Naslov HR:
### Web-aplikacija za digitalno upravljanje narudžbama u ugostiteljstvu primjenom komunikacije u stvarnom vremenu

## Naslov ENG:
### Web application for digital order management in hospitality using real-time communication

## Tekst zadatka:
Načiniti kratki pregled popularnih aplikacija za digitalno upravljanje narudžbama u ugostiteljstvu te predstaviti ključne značajke jednog odabranog rješenja.

Dizajnirati i implementirati oglednu web-aplikaciju kojom se klasični proces posluživanja u ugostiteljskom objektu digitalizira i automatizira primjenom suvremenih web tehnologija. Nakon skeniranja QR koda na mjestu posluživanja, gost pomoću aplikacije i korištenjem personaliziranog digitalnog jelovnika (izbor hrane, izbor pića, itd.) samostalno sastavlja narudžbu, birajući dostupne varijante i dodatke.

Radni proces osoblja obuhvaća preuzimanje, pripremu i isporuku narudžbe, pri čemu je svaki korak sinkroniziran s korisničkim sučeljem gosta. Sustav treba podržati i završnu fazu pružanja usluge, u kojoj gost upućuje zahtjev za finalizaciju i naplatu računa. Cjelokupni životni ciklus narudžbe treba biti popraćen dvosmjernom razmjenom poruka u stvarnom vremenu između svih dionika procesa.
